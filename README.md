# Solstice Cipher

Solstice Cipher is a small browser puzzle game for the DEV June Solstice Game Jam.

The player rotates mirrors to route a solstice beam through cipher nodes and into a beacon. The theme combines the June solstice, light versus darkness, turning points, and an Alan Turing-inspired code-breaking layer.

## Play

Open `index.html` in a browser.

Controls:

- Click or tap mirrors to rotate them.
- `R` resets the current level.
- Arrow keys move between levels.
- `H` shows the level hint.

## Challenge Fit

- Main prompt: a playable game inspired by the June solstice.
- Optional category: Best Ode to Alan Turing.
- Mechanics: binary gates, cipher routing, signal completion, and a small light-path puzzle structure.
